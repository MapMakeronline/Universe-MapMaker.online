## ğŸ¨ Key Design Patterns

### 1. Optimistic Updates
```typescript
// UI updates BEFORE backend confirmation
dispatch(action());
await backendCall();
// If error: rollback
```

**Benefits:**
- Instant UI feedback
- No loading spinners for simple actions
- Better perceived performance

### 2. RTK Query Cache Invalidation
```typescript
invalidatesTags: [
  { type: 'Layers', id: projectName },
  { type: 'Layers', id: 'LIST' },
]
```

**Benefits:**
- Automatic refetch after mutations
- No manual state management
- Always shows latest data

### 3. Error Handling with Rollback
```typescript
try {
  dispatch(optimisticUpdate());
  await backendCall();
} catch (error) {
  dispatch(rollback());
  showError();
}
```

**Benefits:**
- UI always in sync with backend
- User-friendly error messages
- No data loss
